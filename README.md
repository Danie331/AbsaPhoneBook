# AbsaPhoneBook

Requirements:
VS 2019,
.NET Core 3.1,
Docker for Windows,
Node js

Setup:
1. Clone locally
2. Ensure Absa.API set to launch under the Docker profile (should open up Swagger by default)
3. Start separate debug instances of Absa.API and Absa.PhoneBookApplication
